Smart SAVE(구. 하나KS) 안내
신청 안내
  • Smart SAVE 서비스는 할인이 아닌 갚아야 할 부채입니다. 평소 신용카드 이용액을 고려하여 적정한 금액만 이용하시기 바랍니다.
  • Smart SAVE 서비스를 사용하시면 수수료(이자)가 부과되며, 적립된 포인트가 부족할 경우 현금으로 상환하셔야 합니다.
  • Smart SAVE 서비스를 여러 카드사에서 이용하지 마세요.
  • Smart SAVE 서비스를 이용하시면 다른 카드 사용이 제한될 수 있습니다.
이용 안내
  • 하나BC카드 이용고객은 2011년 9월 21일 이후 승인금액부터 이용 가능
  • Smart SAVE 대상카드 추가 발급 고객의 경우 결제일 2영업일 전까지 카드 미 수령 시에 서비스 신청이 취소됨
  • 서비스 이용은 본인회원 당 1건만 가능 (상환 후 재 신청 가능)
  • 분할상환 개월 수는 Smart SAVE 서비스 이용금액에 따라 결정
  • Smart SAVE 취소는 신청 당일만 가능 (당일 이후 취소는 신청한 Smart SAVE 금액(수수료 포함)을 전액 일괄 상환)
  • Smart SAVE 서비스 이용금액은 신청일이 속한 월의 2개월이 경과된 월에 1회 차 청구
  • Smart SAVE 포인트 적립시점부터 기존의 카드서비스는 중단(무이자, 타 포인트 적립 등)
  • 단, Smart SAVE 서비스 상환(만기상환, 선결제, 취소 등)이 완료되면, 다음날부터 기존의 카드서비스로 복원
    • 다 카드 보유 시 : Smart Point 카드, Touch S plus 카드 보유
      Smart SAVE 서비스 신청→ Smart Point 카드의 기존 서비스 중단 / Touch S plus 카드의 기본서비스 가능
포인트 적립 안내
  • 주유/마트/통신/외식 이용금액은 월 적립한도까지만 적립
  • Smart SAVE 포인트 적립 제외 대상
    • Smart SAVE 해당 매출액, 해외 이용금액, 국세/지방세, 연회비, 무이자할부 이용금액, 단기카드대출(현금서비스), 장기카드대출(카드론), 상품권 구매, 대학등록금, 기타제수수료
  • 상환하실 포인트가 부족할 경우 부족한 만큼 매월 현금으로 청구
  • 당월 차감되고 남은 포인트는 이월
    • 예) Smart SAVE 신청금액 : 500,000원 / 당월 적립된 Smart SAVE 포인트 17,800점
      당월 Smart SAVE 상환 포인트(15,365점) - Smart SAVE 적립 포인트(17,800점) = 2,435점 (잔여포인트 이월)
