T SAVE 안내
신청 안내
  • T SAVE 서비스는 할인이 아닌 갚아야 할 부채입니다. 평소 신용카드 이용액을 고려하여 적정한 금액만 이용하시기 바랍니다.
  • T SAVE 서비스를 사용하시면 수수료(이자)가 부과되며, 적립된 포인트가 부족할 경우 현금으로 상환하셔야 합니다.
  • T SAVE 서비스를 여러 카드사에서 이용하지 마세요.
이용 안내
  • T SAVE 서비스 이용은 본인회원 당 1건만 가능하며, 상환완료 시 재신청 가능
  • 회원당 포인트 연계 할부 서비스(Smart SAVE) 및 하나BC Save를 포함하여 1건만 가능
  • T SAVE 서비스 신청 취소는 신청 당일만 가능
  • 최초 상환일은 이용일로부터 30일 후 도래하는 첫 결제일에 청구
  • 상환금액대비 포인트가 부족할 경우 현금으로 청구되며, 차감 후 잔여포인트는 이월됨
  • T SAVE 포인트 적립 시점부터 T SAVE 서비스를 신청한 카드의 모든 서비스(무이자, 타 포인트 적립 등)는 중단됨
  • 단, T SAVE 서비스 상환(만기상환, 선결제, 신청취소 등)이 완료되면 다음 영업일부터 기존의 카드서비스로 복원됨
  • 2개 이상의 하나카드를 보유 시 T SAVE 서비스를 신청한 카드의 서비스만 중단되며, 그 외 카드의 기본서비스는 유지됨
