안내

Priority Pass 카드 안내
  • 고객님의 필요에 따라 연 1회 Priority Pass 카드 신청이 가능합니다.
  • Priority Pass 카드 신청에 따라, 제휴연회비 5만원(연회비 별도)이 부과됩니다.
  • Priority Pass 카드는 본인회원만 이용 가능하며, 본인 회원만 신청 가능합니다.
  • 제휴연회비 : 5만원 (연회비 별도)
  • Priority Pass 서비스는 전세계 Priority Pass 제휴 공항 라운지 무료 이용 시에 제시할 수 있는 별도의 카드를 제공합니다.
  • Priority Pass 서비스는 연 1회 신청 가능하며, 서비스가 해지되는 경우, 신청일 기준으로 1년 경과 후 재신청 가능합니다.
  • 최초 서비스 신청일부터 다음달 말일까지는 지난달 실적에 관계 없이 서비스 제공합니다.
  • Priority Pass 카드 신청 및 무료 입장은 본인회원만 가능하며, 가족회원 명의의 Priority Pass 카드 발급은 불가합니다.
    (본인 회원의 Priority Pass 카드로 가족 회원 동반 입장 시 가족 회원의 라운지 이용요금은 청구됩니다.)
  • 프리미엄 적립 Pack 제휴연회비는 서비스 신청일 기준으로 1년 단위로 청구되며, 서비스 제공 1년이 도래하기 전에 서비스(카드)를 해지하는 경우 회원이 서비스(카드)를 해지한 날부터 잔여기간에 대하여 일할 반환 해드립니다.
  • Priority Pass 서비스는 하나멤버스 1Q 내맘대로 회원에 한하여 추가로 이용할 수 있는 부가서비스이므로 카드를 해지하는 경우 Priority Pass 서비스도 자동 해지됩니다.
  • 회원의 카드 해지 시점에 연회비와 Priority Pass 서비스 잔여기간이 상이할 수 있습니다.
