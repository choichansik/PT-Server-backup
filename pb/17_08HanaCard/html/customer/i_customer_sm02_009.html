웹접근성 정책안내

  • 웹접근성 안내
    웹접근성(Web Accesibility)이란?
    • 웹사이트 이용에 있어 장애에 구애 없이 모든 사람들이 동등하게 웹사이트 정보를 공유할 수 있도록 만드는 것입니다.
    하나카드 웹접근성 준수
    • 하나카드웹사이트는 2010년 12월 국가표준(KICS)으로 제정된 한국형 웹 콘텐츠 접근성 지침 2.0(KICS.OT-0003R1)을 준수하여
      제작하였습니다.
    • 하나카드웹사이트 제작 시 웹 접근성 준수 여부를 점검하기 위해 한국정보화 진흥원에서 개발한 웹 접근성 자동평가도구인 K-WAH4와
      페이지별로 수동평가를 실시하였으며 신규 컨텐츠 추가 및 변경 시 지속적인 점검을 수행하도록 하겠습니다.
    • 하나카드웹사이트는 화면낭독 프로그램 등 보조기기를 이용하여 서비스를 이용하실 수 있도록 개발하였으나, 모든 보조기기가 웹 접근성 지침을 제대로 해석하여 작동되지 않음으로, 이용하시는 보조기기제품에 따라 다소 문제가 발생할 수 있음을 알려드리며, 이는 웹 사이트 제작의 문제가 아니라, 보조기기 제품의 문제임을 알려 드립니다.
    • 하나카드웹사이트에서 제공되는 첨부문서는 PDF, 워드파일 문서형태로 제공됨을 알려 드리며, 모든 첨부문서 내용을 텍스트화 하여 제공하지
      못함을 사이트 이용자에게 알려 드립니다.
    • 하나카드웹사이트는 장애에 구별없이 모든 이용자분들이 불편 없이 웹사이트 정보를 이용할 수 있도록 웹 접근성 준수 여부를 지속적으로
      점검하고 개선하도록 노력하겠습니다.
    하나카드 웹접근성 안내 및 문의처
    • 하나카드웹사이트 이용에 있어서 불편한 부분이 있으시면 언제든지 아래 연락처로 연락 주시기 바랍니다. 항상 신속하게 개선할 수 있도록 최선을 다하겠습니다.
    • 하나카드 고객센터 1800-1111
  • 웹접근성 개선 항목
    • 하나카드웹사이트에서 제공하는 개인/기업/가맹점 웹 서비스에 대해 한국형 웹 콘텐츠 접근성 지침2.0(KICS.OT-0003R1)의 ‘22개의 검사항목’에 대해 전문가 평가 및 장애인 사용자 평가를 진행하여 개선하였습니다.
    웹접근성 개선 항목
    선택 검사항목 설명
    1 적절한 대체 텍스트 제공 이미지 글 텍스트가 아닌 콘텐츠를 이용할 경우 그 의미나 용도를 해당 이미지와 동등하게 인식할 수 있도록 대체 텍스트를 제공
    2 자막제공 멀티미디어 콘텐츠인 동영상, 플래시, 음성을 이용할 경우 장애인도 비장애인과 동등하게 인식할 수 있도록 자막을 제공
    3 색에 무관한 콘텐츠 인식 콘텐츠에서 제공하는 모든 정보는 특정한 색을 구별할 수 없는 사용자나 흑백 디스플레이 사용자, 흑백 인쇄물을 보는 사용자가 색을 배제하여도 해당 콘텐츠를 인식할 수 있도록 제공
    4 명확한 지시사항 제공 특정 요소를 가르키거나 지시사항을 전달하는 콘텐츠에 한정해 적용하는 것으로, 시각 이나 청각 등과 같은 특정 감각에만 의존하여 제공해서는 안됨, 다른 감각을 통해서도 지시사항을 인식하는데 문제가 없도록 제공
    5 텍스트 콘텐츠의 명도대비 웹 페이지에서 보이는 핵심 텍스트 콘텐츠와 배경간의 충분한 명도 대비를 제공하여, 저시력자, 색각 이상자, 고령자 등도 콘텐츠를 인식할 수 있도록 제공
    6 배경음 사용 금지 웹 페이지에서 자동으로 재생되는 배경음이 있는 동영상, 음성, 음악등으로 인해 낭독 프로그램을 사용하는 사용자가 콘텐츠를 인식하는데 방해를 받지 않아야 함
    7 키보드 사용 보장 웹페이지에서 제공되는 모든 기능을 사용자가 키보드만으로도 사용할 수 있도록 제공
    8 초점 이동 키보드 내비게이션 과정에서 키보드 초첨을 받은 링크, 컨트롤 및 입력 서식은 초첨을 받지 않은 객체들로부터 구분 될 수 있어야 함
    9 응답시간 조절 웹 콘텐츠 제작 시 시간제한이 있는 콘텐츠는 가급적 제공하지 않는 것이 바람직하며, 보안 등의 사유로 시간 제한이 필요한 경우 시간적인 제약이 있음을 사용자에게 반드시 알려주어야 하며, 예외의 경우를 제외한 모든 콘텐츠는 시간제약을 회피하거나 우회할 수 있는 수단을 함께 제공
    10 정지 기능 제공 웹 콘텐츠에서 스크롤 및 자동 갱신되는 콘텐츠를 장애인 사용자가 이용할 수 있도록 일시 정지 할 수 있는 방법을 제공함, 저시력자나 지적장애인 등은 이동하거나 스크롤 되는 콘텐츠를 인지하기 어렵기 때문에 '앞으로이동', '뒤로이동', '정지'등의 콘트롤을 제공
    11 깜빡임과 번쩍임 사용 제한 깜빡임과 번쩍이는 콘텐츠는 간질병 등의 초래 가능성이 있으므로 초당 3~50회의 주기로 깜빡이거나 번쩍이는 콘텐츠를 제공하지 않아야 함
    12 반복 영역 건너뛰기 화면 낭독 프로그램을 이용하는 사람들은 반복되는 메뉴 등을 페이지 마다 다시 듣게 됨으로, 이런 불편을 막기 위해 메뉴 등과 같이 페이지마다 공통되며 반복되는 영역을 사용자가 바로 건너뛰어 핵심 콘텐츠로 이동할 수 있도록 건너뛰는 기능을 제공함
    13 제목제공 페이지, 프레임, 콘텐츠 블록에 적절한 제목을 제공하여 웹 콘텐츠를 운용하기 쉽고 제목은 간단명료해야 하며 해당 페이지, 프레임, 콘텐츠 블록을 유추할 수 있도록 제공
    14 적절한 링크 텍스트 링크의 용도나 목적지를 명확하게 이해할 수 있도록 링크 텍스트를 제공
    15 기본 언어 표시 텍스트 콘텐츠의 언어 정보를 화면 낭독 프로그램으로 제공하여 정확한 발음이 가능하도록, 화면 낭독 프로그램을 제어하기 때문에 웹 페이지를 구성하는 기본 언어는 정확히 명시해야 함
    16 사용자 요구에 따른 실행 사용자가 예측할 수 없는 상황에서 정보가 제공되지 않도록, 서식 컨트롤이나 사용자 압력이 초점을 받았을 때 의도하지 않는 기능이 자동적으로 실행되지 않아야 함
    17 콘텐츠의 선형화 콘텐츠 보조 기기 등을 통해서도 논리적인 순서로 이해할 수 있도록 제공
    18 표의 구성 표로 제공할 경우, 시각장애인 등도 쉽게 이해할 수 있도록 이해를 돕기 위한 내용 및 구조에 대한 정보 등을 제공
    19 레이블 제공 입력 서식을 사용할 경우, 시각 장애인 등이 해당 서식을 이해할 수 있도록 레이블을 제공
    20 오류 정정 입력 서식 작성 시, 사용자의 실수로 잘못 오류가 발생할 경우 이를 정정할 수 있는 방법을 제공
    21 마크업 오류 방지 마크업 언어의 요소를 사용할 경우, 해당 마크업의 문법을 최대한 준수하여 태그의 열고 닫음, 중첩 관계의 오류가 없도록 제공
    22 웹 애플리케이션 접근성 준수 웹 콘텐츠에 포함된 부가 애플리케이션 또는 웹 페이지의 기능을 실행하는데 필요한 웹 어플리케이션은 웹페이지를 사용하거나 접근하는 것을 방해하지 않도록 반드시 사용자의 접근성을 지원해야 하며 보조기술을 이용하여 사용자가 접근 할 수 있는 대체수단을 함께 제공함
  • 이용 가이드
    하나카드웹사이트의 접근성 한계점
    • 하나카드사이트는 자체적으로 제작한 콘텐츠에 웹접근성을 보장하기 위해 노력하고 있습니다, 다만, 제작 콘텐츠가 아닌 외부로 부터 제공받은 콘텐츠 및 일부 페이지, 서비스는 장애인 여러분들께서 이용하시는데 어려움을 겪으실 수 있습니다.
    • 오래된 콘텐츠의 접근성을 보장하기 위해 개선 작업을 진행하고 있으나, 일부 콘텐츠에 대해서만 웹 접근성을 제공하게 되어, 사용에 불편을 초래할 수 있는 점 양해 부탁 드립니다. (하나카드사이트에서는 2013년1월 이후 게시물에만 웹접근성을 준수하는 정책을 적용하고 있습니다.)
    스크린리더
    • 시각장애인 여러분들께서 하나카드웹사이트를 이용하시려면 화면을 읽어주는 별도의 스크린리더 프로그램이 필요하며, 스크린 리더 프로그램이 없으신 고객은 유료로 판매되는 제품을 구입하거나 실로암 사회복지과내(http://www.silwel.or.kr, 연락처 : 02-8800-840~5)에서 시각장애인에 한하여 무료로 배포하는 ‘드림보이스’ 프로그램을 설치하면 이용이 가능합니다, 이를 이용한 홈페이지 이용 중에 발생하는 문제는 보조기기들의 특성에 따라 다를 수 있으므로 이용하시는 고객들께서는 참고하여 주시면 감사하겠습니다.
    뷰어 프로그램 다운로드
    • 하나카드웹사이트에서 운영되는 콘텐츠는 PDF, 워드(DOC)등의 문서가 제공되고 있으며 또한 해당 파일을 열람 가능하도록 뷰어 를 다운받아 이용하실 수 있도록 관련 프로그램의 다운로드 기능을 제공하고 있습니다, 단. 제공되는 뷰어 프로그램은 각 제작사의 운영 정책에 따라 단순 링크만이 가능하므로 아래의 링크를 클릭하여 이동 후 다운받아 설치하여야 합니다.
