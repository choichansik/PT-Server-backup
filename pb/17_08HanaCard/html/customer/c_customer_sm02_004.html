이용안내보안프로그램 장애해결

보안프로그램 장애해결

  • 통합보안프로그램 설치안내
    하나카드 통합보안프로그램 설치안내
    • 고객님의 안전한 웹서비스 이용을 위해 하나카드 홈페이지 처음 접속 시 Veraport 통합보안프로그램이 자동으로 설치됩니다.
    • 통합보안프로그램이 정상적으로 설치되지 않으면 하나카드 홈페이지의 일부 서비스 이용이 제한될 수 있습니다.
      • 금융감독원 지시사항으로 하나카드 홈페이지 이용 시 암호화/키보드보안/개인방화벽 프로그램을 의무적으로 설치하도록 하고 있습니다.
    통합보안프로그램 자동설치 재시도
    • 하나카드 홈페이지 처음 접속 시 Veraport 통합보안프로그램이 자동으로 설치됩니다.
    • 약 10초정도(통신환경에 따라 다름) 기다리시면 자동으로 설치되지만, 설치가 완료되기 전에 마우스로 [중지]를 누르신 경우 설치가 진행되지 않습니다. 이 경우 스크립트 오류가 발생하여 정상적으로 하나카드 홈페이지를 이용할 수 없습니다.
    • 보안프로그램을 다시 설치하기 위해서는 첫 페이지에서 [새로고침] 아이콘을 누르거나 [Ctrl+F5]키를 2~3번 눌러주면, 페이지가 다시 시작되면서 보안프로그램이 설치됩니다. 중간에 보안경고창이 나오면 반드시 [예]를 선택하십시오.
    • 만약 그래도 보안프로그램이 정상적으로 설치되지 않는 경우, 다음 단계의 조치사항을 시행해 주시기 바랍니다.
  • Active X 컨트롤 설치안내
    • Active X컨트롤을 설치하셔야 보안프로그램이 자동으로 설치됩니다.
    • 아래와 같이 Active X컨트롤을 설치하신 후, 보안프로그램을 설치하시기 바랍니다.
    01 주소창 아래 보여지는 Active X 컨트롤 설치 안내문구를 클릭!
    02 메뉴창에서 Active X 컨트롤 설치를 선택
    03 보안경고창(암호플러그인 설치)이 뜨면 설치를 선택
    위의 방법으로 안되신다면!
    • Active X 컨트롤 설치 안내문구가 확인되지 않거나 설치가 정상적으로 안되는 경우 아래와 같이 시행해 보시기 바랍니다.
  • 보안프로그램 장애해결
    임시 인터넷파일 삭제와 인터넷 옵션 변경 방법
    • 01.우선 사용중인 Internet Explorer창 닫기
    • 02.윈도우 화면 하단의 [시작] 버튼을 클릭하여 [설정 > 제어판 > 인터넷옵션] 선택
    • 03.[파일삭제] 선택 후 [오프라인항목 모두 삭제] 항목에 체킹 후 확인버튼 클릭
    • 04.파일 삭제 작업이 끝나면 다시 [설정] 선택 후 페이지를 열 때마다에 체킹하고 확인 버튼을 클릭
    • 05.다시 [인터넷 옵션] 창 상단 메뉴 중 [보안]을 선택하고, 아래쪽에 사용자지정수준 버튼을 클릭하여 원래대로 선택하고 보안설정 변경 물음에 [예]를 클릭한 후 확인 버튼 클릭
    • 06.다시 [인터넷 옵션] 창 [보안] 메뉴창으로 돌아와 하단의 기본수준 선택
    • 07.다시 [인터넷 옵션] 창 상단 메뉴 중 [고급]을 선택하여 아래쪽에 기본값 복원을 클릭한 후 적용 버튼 클릭
    • 08.이번에는 인터넷 옵션창을 종료하시고 윈도우 화면 하단의 [시작] 버튼을 클릭하여 [(설정)>제어판>프로그램추가/제거]를 선택
    • 09.INIPlugin 4.0, INISafeWeb 5.0 / 6.0 / 7.0,INISafe Sfilter 7.2를 선택하여 삭제 (일부 프로그램만 설치되어 있을 경우 해당 프로그램만 삭제)
    • 10.Internet Explorer를 다시 실행시키고 홈페이지 주소 www.hanacard.co.kr을 직접 입력해야 실행합니다.
      보안경고창이 뜨는 경우에는 [예]를 누르시면 설치가 진행됩니다.
      (보안경고창이 뜨지 않는 경우는 자동적으로 설치 진행)
  • 보안프로그램 수동설치 방법
    하나카드 통합보안프로그램 설치안내
    • 위와 같은 방법으로 인터넷 옵션을 최적화하신 후 아래의 보안 관련 프로그램을 수동으로 직접 PC에 다운받아 인터넷익스플로러 브라우저를 모두 닫은 후, 다운받은 프로그램을 더블클릭하여 설치한 다음, 다시 www.hanacard.co.kr 주소를 직접 입력하여 접속해 보시면 대부분 정상 이용이 가능합니다. 보안프로그램 다운로드
    • 보안프로그램 다운로드 버튼은 모두 설치되어 있으면 동작하지 않습니다.
    • 만약, 위와 같은 방법을 모두 시행하였는데도 불구하고 정상적인 이용이 안되실 경우, 하나카드 고객센터(1800-1111)로 문의해 주시기 바랍니다.
