모바일 공인인증서 복사

공인인증서 내보내기 PC→스마트폰
  • PC에서 발급된 공인인증서를 고객님의 스마트폰으로 내보내기 할 수 있습니다.
    • 은행에서 발급한 공인인증서 또는 범용공인인증서 사용 가능
공인인증서 내보내기
공인인증서 가져오기 스마트폰→PC
  • 스마트폰의 공인인증서를 PC로 가져오기 할 수 있습니다.
    • 은행에서 발급한 공인인증서 또는 범용공인인증서 사용 가능
공인인증서 가져오기
  • 1스마트폰에서 공인인증서 관리 > 인증서 내보내기 메뉴 선택
  • 2PC에서 공인인증서 선택 후 비밀번호 입력
  • 3PC에 표시된 인증번호 8자리를 스마트폰에 입력
  • 4스마트폰에서 공인인증서 비밀번호 입력
  • 5PC에서 스마트폰으로 공인인증서 이동 (완료)
  • 1스마트폰에서 공인인증서 관리 > 인증서 내보내기 메뉴 선택
  • 2스마트폰에서 공인인증서 선택 및 비밀번호 입력
  • 3스마트폰에 표시된 인증번호 8자리 표시 확인
  • 4PC의 공인인증서 가져오기 버튼 선택 후, 스마트폰에 표시된 인증번호 6자리 입력
  • 5PC에서 공인인증서 비밀번호 입력
  • 6스마트폰에서 PC로 공인인증서 이동 (완료)

공인인증서 내보내기/가져오기 안내

공인인증서 내보내기/가져오기 안내
  • 하나카드 스마트폰 어플리케이션은 고객의 안전한 금융거래를 위해 공인인증서 로그인 기능을 제공합니다.
  • PC에서 발급된 공인인증서를 고객님의 스마트폰으로 내보내기 할 수 있으며, 스마트폰의 공인인증서를 PC로 가져오기 할 수 있습니다.
    (은행에서 발급한 공인인증서 또는 범용공인인증서 사용 가능)
